Ucmatose Test
-------------
RDMA CM connection and simple ping-pong test.

Prerequisites
-------------
MOFED installed.
Passwordless SSH enabled to peer.
Python module 'netifaces' is needed. Install via 'pip install netifaces (or) easy_install netifaces'

Input Needed
------------
ext_flag        -   Flag to indicate whether to run test with extended parameters or not(1 - yes, 0 - no)
Iface           -   Interface
PEERIP          -   Peer IP Address
basic_option    -   parameter for basic test
ext_option      -   parameter for extended test
